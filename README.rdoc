== SprintTracker

SprintTracker is an application that allows users to track their sprint todos. Users can easily manage story priority
and check their burndown. The backend is using Ruby on Rails and the front end is using Backbone.js

==== Technologies Used

Backbone.js framework used to manage views within the single page app

jQuery UI used to handle drag and drop and sorting of stories

HighCharts js library used for burndown chart generation
